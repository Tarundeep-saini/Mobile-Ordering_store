# Mobile-Ordering_store
Made using MERN , Tailwind CSS

This Is an E-Commerce store for Buying and Selling Mobiles.


## Installation

# API
Install the dependencies and devDependencies and start the server.

```sh
cd API
npm install
npm start
```


# API
Install the dependencies and devDependencies and start the server.

```sh
cd Client
npm install
npm run dev
```
